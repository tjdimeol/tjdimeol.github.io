Project Overview

This project explores diffusion models from first principles, implementing forward noising processes, reverse denoising processes, and full iterative sampling algorithms using DeepFloyd IF. The work includes understanding the mathematical foundations of diffusion models, implementing classifier-free guidance, creating visual anagrams, and training custom diffusion models on MNIST from scratch.
